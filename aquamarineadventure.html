Aquamarine Adventure

“Aquamarine Adventure” is an interactive scavenger hunt game made using Javascript, HTML and CSS during the Web Narratives class at UCSB. This web design follows Whiskers the cat as she ventures down below to visit her trusty friend Capy. Each 'level' in the game depicts a different underwater themed background and new icons to scavenge in order to help Whiskers along her journey. Once the end is reach there is exit scene where the characters meet!
